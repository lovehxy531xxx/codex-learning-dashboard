# Codex 90 天学习与交付驾驶舱

这是一个本地静态网页工具，用来记录 Codex 学习、工具开发、客户沟通、需求发现和每日复盘。

## 使用

直接打开 `index.html` 即可使用。

数据保存在浏览器的 `localStorage` 中。建议定期在“备份”页面下载 JSON 备份文件。

如果要让其他人通过公网访问，请看 `DEPLOY.md`。本仓库已经包含 GitHub Pages 自动部署工作流。

## 第一版功能

- 90 天路线图
- 每天具体行动清单和分时段打卡
- 一级“每日计划”页面，可查看 90 天全部计划
- 每个计划项支持笔记和心得
- 每日学习和工作记录
- 学习、做工具、需求研究、找客户、发内容、复盘打卡
- 累计小时、工具产出、用户触达、内容发布统计
- 需求卡记录
- 每日报告复制
- 今日复盘提示词复制
- 学习资料库
- 常用 Codex 提示词库
- JSON 导出和导入
